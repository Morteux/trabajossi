# Instalar IPFS
https://ipfs.io/

### Primera forma
Descargar el .exe de Drive
Descomprimirlo
Mover **ipfs.exe** a **%PATH%**.

Para probar que funciona:
```
$ ipfs help
USAGE:

    ipfs - Global p2p merkle-dag filesystem.
...
```

### Segunda forma
Ejecutar el .exe de la aplicación de escritorio.
